# Space Engineering Skills for AI

This repository is created to help with space engineering by providing different specialized skills for AI agents. Design satellite & on-surface missions from concept to orbit.


## Getting Started

To manage, install, and use these skills, we suggest using [`npx skills`](https://skills.sh/).

### Folder Structure

- **External Skills**: Skills that you install or import are stored in the `.agents/skills` folder.
- **Local Skills**: Skills created directly within this repository are stored in the `skills` folder.
